## Instructions on how to run the project locally  
In backend  
`mpn init`  
`npm start`  
You should get "Math Masters Backend is running!" on http://localhost:5000/  

In frontend as well  
`npm init`  
`npm start`  

In the .env file, you can replace my database credentials with yours to gain access.
